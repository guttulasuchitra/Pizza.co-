# 🍕 Pizza.co-

A sleek and modern pizza menu application built with React.

🚀 Overview

pizza.co is a user-friendly platform that dynamically displays a menu of delicious pizzas, complete with details such as ingredients, prices, and availability. The app uses React's component-based architecture to create an engaging and interactive user experience, perfect for demonstrating modern web development practices.

🌟 Features

Dynamic Menu Display: Showcases a list of pizzas with their details, including name, ingredients, price, and sold-out status.
Responsive Design: Ensures optimal usability on both desktop and mobile devices.
Conditional Rendering: Displays different content based on the restaurant's open hours and pizza availability.
Reusable Components: Modular design with reusable components like Header, Menu, Pizza, and Footer.
Interactive Elements: Includes a call-to-action button for ordering pizzas.

🛠️ Technologies Used

Frontend: React, CSS Modules

Development Tools: Node.js, npm
